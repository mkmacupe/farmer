# Product Image Requirements

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Format | WebP |
| Size | 400x400 px (square) |
| File size | 30-50 KB max |
| Background | White (#FFFFFF) or light gray (#F5F5F5) |
| Style | Professional product photography, centered |
| Lighting | Soft, even studio lighting |
| Shadow | Subtle drop shadow allowed |

## Design Guidelines

1. **Consistency**: All images must have the same visual style
2. **Centered**: Product occupies 70-80% of frame, centered
3. **No text overlays**: Product only, no labels or watermarks
4. **High quality**: Sharp, well-lit, professional appearance
5. **Realistic colors**: True-to-life color representation

## Quality Checklist

Before adding each image, verify:

- [ ] Format is WebP
- [ ] Size is approximately 400x400 px
- [ ] File size under 50 KB
- [ ] White or light gray background
- [ ] Product is centered
- [ ] No text/watermarks/logos
- [ ] Professional quality
- [ ] Unique (not duplicate of another product)
- [ ] Matches product description
